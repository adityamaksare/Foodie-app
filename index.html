MERN Food Delivery Application Index

Overview

This is a full-stack food ordering web application built using the MERN stack (MongoDB, Express, React, Node.js). The application consists of three main parts:

  • Frontend: Customer-facing app for ordering food
  • Backend: API server handling data and business logic
  • Admin Panel: Interface for managing orders, menu items, and more

1. Backend Structure

Main Components

server.js: Entry point that sets up Express server, connects to MongoDB, and defines API routes

Models

  • foodModel.js: Schema for food items
  • userModel.js: Schema for user accounts
  • orderModel.js: Schema for customer orders

Controllers

  • foodController.js: Handlers for food-related operations
  • userController.js: Handlers for user authentication and profile management
  • cartController.js: Handlers for shopping cart operations
  • orderController.js: Handlers for order processing including payment integration

Routes

  • foodRoute.js: API endpoints for food items
  • userRoute.js: API endpoints for user operations
  • cartRoute.js: API endpoints for cart management
  • orderRoute.js: API endpoints for order handling

API Endpoints

Route Description
/api/food Food item CRUD operations
/api/user User registration, login, profile management
/api/cart Shopping cart operations
/api/order Order processing and payment
/images Static file serving for uploaded images

2. Frontend Structure

Main Pages

  • Home: Landing page with food listings
  • Cart: Shopping cart management
  • PlaceOrder: Checkout and payment processing
  • MyOrders: Order history and tracking
  • Verify: Authentication related pages

Key Features

  • User authentication (login/register)
  • Browse food catalog
  • Add items to cart
  • Checkout with Stripe payment integration
  • Order tracking
  • Responsive design

3. Admin Panel Structure

Main Pages

  • Add: Add new food items to the menu
  • List: View and manage existing food items
  • Orders: Track and manage customer orders

Key Features

  • Food item management (CRUD operations)
  • Order processing workflow
  • Order status updates
  • Admin authentication

4. Technology Stack

Component Technologies
Frontend React.js, React Router, Context API, Vite
Backend Node.js, Express.js, MongoDB, Mongoose
Authentication JWT (JSON Web Tokens)
Payment Processing Stripe
Styling CSS

5. Running the Application

Backend

cd backend && npm install && npm run server

Runs on: http://localhost:4000

Frontend

cd frontend && npm install && npm run dev

Runs on: http://localhost:5173

Admin Panel

cd admin && npm install && npm run dev

Runs on: http://localhost:5174
